1. Install Flutter framework at https://docs.flutter.dev/get-started/install
2. To create a new project, open the terminal and use "flutter create <project-name>"
3. Replace the files inside the project with the files in the repo folder.
4. Run the project using "flutter run" and choose your desired emulator.
5. If a null sound safety error occurs, proceed with "flutter run --no-sound-null-safety"
6. If a web image renderer error occurs, proceed with "flutter run --web-renderer=html"
